# vue-build-tools-exercise

A simple Vue.js project to exercise build tools. Learning Vue.js through Frontend Masters course [Introduction to Vue.js](https://frontendmasters.com/courses/vue/).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Run your end-to-end tests

```
npm run test:e2e
```

### Lints and fixes files

```
npm run lint
```
